## Обновление версии Unity

### Описание проблемы
Обнаружено несоответствие между сохраненной версией проекта (2022.3.0f1) и запущенным редактором Unity (2022.3.62f3).

### Решение
Обновлена версия Unity в файле ProjectSettings/ProjectVersion.txt с 2022.3.0f1 до 2022.3.62f3, чтобы устранить конфликт версий.

### Измененные файлы
- ProjectSettings/ProjectVersion.txt

### Тестирование
Рекомендуется протестировать проект после слияния, чтобы убедиться, что все функции работают корректно с новой версией редактора.

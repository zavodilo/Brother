Materials for RimWorld-Style Game

This folder should contain materials used in the project:

1. TerrainMaterial.mat - Material for the procedural terrain
   - Shader: Standard or Terrain shader
   - Albedo texture: Grass, dirt, rock textures
   - Splat textures for terrain blending

2. CityIconMaterial.mat - Material for city icons
   - Shader: UI/Default or Unlit/Color
   - Color: Distinct color to stand out on terrain

3. BuildingMaterial.mat - Material for city buildings
   - Shader: Standard
   - Colors: Various colors for different building types

These materials should be created in Unity and assigned to the appropriate objects.
